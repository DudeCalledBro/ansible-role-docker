# Ansible Role: Docker

> An Ansible Role that installs and configures Docker (currently only Debian/Ubuntu).

## Example

The following minmal Playbook will setup and configure unattended upgrades.

```yaml
- hosts: all
  roles:
  - dudecalledbro.docker
```
